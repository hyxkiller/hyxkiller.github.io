flutter常用组件基础

Posted on 2019-02-20

常用组件

  • TextWidget

    • 常用属性
      1. TextAlign
      2. maxLines 最大行数
      3. overflow(fade, ellipsis, clip)

  • ContainerWidget

    • 内边距、外边距
      EdgeInsets.all()统一设置
      EdgeInsets.fromLTRB(v1, v2, v3, v4)
    • decoration修饰器(设置容器边框)
      BoxDecoration()

  • ImageWidget

    • 引入方式
      1. Image.asset 加载资源图片，会使打包体积过大
      2. Image.network 网络资源图片
      3. Image.file 本地图片，相机照相后图片预览
      4. Image.memory 加载到内存中的图片 Uint8List
    • Fit属性
      BoxFit.contain(最大程度拉伸，图片未变形)
      BoxFit.cover(图片保持原比例，可能被裁切)
      BoxFit.fill(宽高撑满，图片变形)
      BoxFit.fitHeight(高度撑满，宽度裁切)
      BoxFit.fitWidth(宽度撑满，高度裁切)
    • 混合模式
      colorBlendMode: BlendMode.xxx
    • 重复
      repeat: ImageRepeat.xxx

  • ListView

  • GridView

    • 属性
      crossAxisSpacing 网格间空隙
      crossAxisCount 网格列数
      childAspectRatio 设置宽高比
      maxCrossAxisExtent 排列每一项最大宽度
    • GridView.count
    • GridView.builder
    • GridView.custom
    • GridView.extent

  • PageView 可横向纵向滚动

    1. onPageChanged事件 可监听当前页面index
    2. pageSnapping 取消滑动跳页功能
    3. scrollDirection 滚动方向

  • AppBar参数

    1. leading 标题前显示一个控件
    2. title 标题文字
    3. actions 标题右侧控件
    4. bottom 一个AppBarBottomWidget对象, 用来在 Toolbar 标题下面显示一个 Tab 导航栏, 对应body的children Widget

  • Drawer 会自动在标题左侧加一个menu图表，点击左侧划出

  • endDrawer 会自动在标题右侧加一个menu图表，点击右侧划出

  • GestureDetector 手势组件

  • RefreshIndicator 下拉刷新组件(onRefresh为必有方法)
