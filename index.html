引入谷歌统计

Posted on 2019-02-20

如何加入谷歌统计

  1. 创建或登录到您的 Google Analytics帐号( google.com/analytics)
  2. 在 Google Analytics（分析）帐号中设置媒体资源。媒体资源代表您的网站或应用。
    如何设置媒体资源？
    • 登录到谷歌分析后，点击左下角管理，在“帐号”列(第一列)中，来选择要向其添加媒体资源的帐号。如果有多个帐号，可使用搜索框找到所需的帐号。
    • 在“媒体资源”列(第二列)中，点击创建新媒体资源。如果您对相应帐号没有“修改”权限，则看不到创建新媒体资源选项。请检查您是否已 从“帐号”列中选择了正确的帐号。
    • 选择网站或移动应用。如果您选择网站，请跳至第 5 步。如果您选择移动应用，请继续执行第 4 步。
    • （移动应用）在“跟踪方法”下，选择 Firebase。从菜单中选择一个应用，然后点击关联应用。
    • 输入网站名称或应用名称。如果打算在帐号中跟踪多个应用，请使用非常具体的描述性名称，并在其中包含版本号。方便日后管理。（仅限网站）输入网站网址。如果网址的格式不正确，就无法创建媒体资源。
    • 选择行业类别。
    • 选择报告时区。此时区将用作报告的每日起止时间，而无论数据来自何处。
    • 点击获取跟踪 ID。
  3. 在您的媒体资源中设置报告数据视图。数据视图支持显示网站的某一部分跟踪数据，过滤掉其他不重要部分的数据。没有创建视图默认是全部网站数据。
  4. 按照说明将跟踪代码添加到您的网站或移动应用。
    如何查找跟踪ID？
    • 登录到您的 Google Analytics（分析）帐号
    • 点击左下角管理
    • 从“帐号”列的菜单中选择一个帐号。
    • 从“媒体资源”列的菜单中选择一个媒体资源。
    • 在“媒体资源”下，点击跟踪信息 > 跟踪代码。跟踪 ID 会显示在页面的顶部。

  • 验证跟踪代码是否正常工作 要验证跟踪代码是否正常工作，请访问您的网站，并检查您的这次访问是否已在实时报告(在左侧面板中，打开实时类别下的概览。)中记录。

flutter常用组件基础

Posted on 2019-02-20

常用组件

  • TextWidget

    • 常用属性
      1. TextAlign
      2. maxLines 最大行数
      3. overflow(fade, ellipsis, clip)

  • ContainerWidget

    • 内边距、外边距
      EdgeInsets.all()统一设置
      EdgeInsets.fromLTRB(v1, v2, v3, v4)
    • decoration修饰器(设置容器边框)
      BoxDecoration()

  • ImageWidget

    • 引入方式
      1. Image.asset 加载资源图片，会使打包体积过大
      2. Image.network 网络资源图片
      3. Image.file 本地图片，相机照相后图片预览
      4. Image.memory 加载到内存中的图片 Uint8List
    • Fit属性
      BoxFit.contain(最大程度拉伸，图片未变形)
      BoxFit.cover(图片保持原比例，可能被裁切)
      BoxFit.fill(宽高撑满，图片变形)
      BoxFit.fitHeight(高度撑满，宽度裁切)
      BoxFit.fitWidth(宽度撑满，高度裁切)
    • 混合模式
      colorBlendMode: BlendMode.xxx
    • 重复
      repeat: ImageRepeat.xxx

  • ListView

  • GridView

    • 属性
      crossAxisSpacing 网格间空隙
      crossAxisCount 网格列数
      childAspectRatio 设置宽高比
      maxCrossAxisExtent 排列每一项最大宽度
    • GridView.count
    • GridView.builder
    • GridView.custom
    • GridView.extent

  • PageView 可横向纵向滚动

    1. onPageChanged事件 可监听当前页面index
    2. pageSnapping 取消滑动跳页功能
    3. scrollDirection 滚动方向

  • AppBar参数

    1. leading 标题前显示一个控件
    2. title 标题文字
    3. actions 标题右侧控件
    4. bottom 一个AppBarBottomWidget对象, 用来在 Toolbar 标题下面显示一个 Tab 导航栏, 对应body的children Widget

  • Drawer 会自动在标题左侧加一个menu图表，点击左侧划出

  • endDrawer 会自动在标题右侧加一个menu图表，点击右侧划出

  • GestureDetector 手势组件

  • RefreshIndicator 下拉刷新组件(onRefresh为必有方法)

flutter环境搭建

Posted on 2019-02-20

环境安装

  1. 官网下载 flutter SDK https://flutter.io/docs/get-started/install/macos
  2. 解压后放到一个目录下
  3. 配置环境变量
    vim ~/.bash_profile
加一项
export PATH=安装路径/bin:$PATH
source ~/.bash_profile
  4. 执行flutter doctor，查看安装情况
  5. 此外还需装java环境，Xcode，Android Studio
  6. 升级命令

    flutter upgrade

VSCode搭建开发环境

  1. 扩展里装flutter，如果出现无法安装按步骤下载.vsix继续安装
  2. flutter emulators查看当前可使用设备
  3. flutter emulators –launch 设备名 启动虚拟机
  4. flutter run 启动项目
  5. 快捷键
    • o 切换Android，ios
    • p 显示网格线
    • r 热加载
  6. 或可打开调试，开启热更新

nginx基础命令

  • 快速停止 kill -TERM 主进程号
  • 强制停止 kill -9 nginx

  • 修改/usr/local/etc/nginx/nginx.conf文件配置

    proxy_pass 后端部署地址;

    proxy_set_header Connection "upgrade";

算法复杂度

Posted on 2019-01-09

复杂度也叫渐进复杂度，包括时间复杂度和空间复杂度，时间复杂度表示执行的快慢，空间复杂度表示执行的消耗

时间复杂度

  • 只关注循环执行最多的一段代码
  • 加法法则：总复杂度等于量级最大的那段代码的复杂度。
  • 乘法法则：嵌套代码的复杂度等于嵌套内外代码复杂度的乘积

  • 常见时间复杂度分析

    • O(1) 常数阶

    • O(logn) 对数阶, O(nlogn)

    • O(m+n), O(m*n)

操作数量事例 大O表示法 术语
10 O(1) 常数阶
2logn O(logn) 对数阶
3n+5 O(n) 线性阶
5n^2 + 2n O(n^2) 平方阶
2^n+1 O(2^n) 指数阶
n!+3 O(n!) 阶乘阶

空间复杂度

表示算法的存储空间和数据规模的关系。

  • 常见的只有O(1), O(n), O(n^2)

思考题

此函数时间复杂度为O(n)，怎么降为O(1)
